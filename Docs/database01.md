## 数据表清单

| --------------- | --------------------------------- |
| **开发环境**    | **MySQL**                         |

| Teacher         | 教师                              |
| Admin           | 管理员                            |

| StudentExam     | 学生考试信息 已参加未参加  总成绩 |
| News            | 公告、通知                        |
| Message         | 学生留言、提问                    |
| Replay          | 教师答复                          |
| AuthorityManage | 权限管理                          |
| Spare           | 备用                              |
